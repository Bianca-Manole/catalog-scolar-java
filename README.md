\# 📚 Catalog Școlar - Java Project



\## 🔹 Descriere



Aplicație Java pentru gestionarea unui catalog școlar.



Permite administrarea elevilor, cursurilor și notelor, precum și calculul mediilor și generarea clasamentelor.



\---



\## 🔹 Funcționalități



\* ➕ Adăugare elevi în clasă

\* 📘 Asociere cursuri elevilor

\* 📝 Adăugare note în catalog

\* 📊 Calcul medie generală elev

\* 📚 Calcul medie pe curs

\* 🏫 Calcul media clasei

\* 🥇 Clasament elevi (sortare descrescătoare)

\* 🥈 Locul 1 și locul 2

\* 👑 Șef de promoție (media > 9.50)

\* 🎖️ Elevi cu mențiune:



&#x20; \* media ≥ 8

&#x20; \* media ≤ media locului 2



\---



\## 🔹 Structura proiectului



\* \*\*Student\*\* – elevul

\* \*\*Curs\*\* – materia

\* \*\*Nota\*\* – nota unui elev

\* \*\*Catalog\*\* – gestionează notele

\* \*\*Clasa\*\* – conține elevii și catalogul



\---



\## 🔹 Tehnologii utilizate



\* Java

\* OOP (Encapsulation, Abstraction)

\* Collections (List, ArrayList)

\* Comparator pentru sortare
